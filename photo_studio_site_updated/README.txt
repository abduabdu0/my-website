README - Обновлённый проект (видео фон, светлая палитра, сохранение заявок в Excel + email)
=========================================================================================

Что сделано
- Светлая "свадебная" цветовая палитра.
- Видеофон на главной (файл: assets/video/neutral.mp4). Если видео нет, будет использоваться poster (assets/images/poster.svg).
- Анимации: простая логика на IntersectionObserver (assets/main.js).
- Форма отправляет POST /submit (JSON) к простому Flask-серверу.
- Сервер сохраняет каждую заявку в submissions.xlsx и отправляет файл Excel на EMAIL_TO через SMTP.

Как запустить фронтенд локально
1. Распакуй архив.
2. Открой index.html в браузере (видеофайл - замените assets/video/neutral.mp4 на реальный MP4).

Как запустить сервер (Python)
1. Установи Python 3.9+ и создайте виртуальное окружение.
2. Установи зависимости:
   pip install -r server/requirements.txt
3. Отредактируй server/app.py: укажи SMTP_HOST, SMTP_PORT, SMTP_USER, SMTP_PASS, EMAIL_TO.
4. Запусти сервер:
   python server/app.py
5. Фронтенд отправит запросы на http://localhost:5000/submit

Важно о безопасности
- SMTP-пароли и данные нужно хранить в безопасном месте (например, ENV переменные). В примере они в файле для простоты — не забудь заменить.
- Сервер в примере не защищён от спама/валидации. Для продакшена добавь CAPTCHA и проверку входных данных.
