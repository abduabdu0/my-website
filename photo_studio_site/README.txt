README - Проект сайта фото/видеостудии
===================================

Структура:
- index.html        - Главная страница
- services.html     - Услуги и пакеты
- portfolio.html    - Портфолио (пример)
- price.html        - Прейскурант / Цена
- contacts.html     - Контакты и карта
- assets/style.css  - Стили
- assets/main.js    - Небольшая JS логика
- assets/images/*   - Заглушки для изображений

Как запустить локально:
1. Распакуй архив.
2. Открой index.html в браузере (двойной клик).
3. Для форм/письма подключи сервис Formspree или свой бэкенд.

Google Maps:
- В contacts.html вставлен iframe-заглушка.
- Чтобы показать свою карту, встрани сюда ссылку с Google Maps Embed:
  <iframe src="YOUR_GOOGLE_MAPS_EMBED_URL" ...></iframe>

Публикация на GitHub Pages:
1. Создай репозиторий на GitHub.
2. Загрузи все файлы (или используй git).
3. В настройках репозитория включи Pages и выбери ветку main / root.
4. Через минуту сайт будет доступен по ссылке вида: https://USERNAME.github.io/REPO

Изменение текста/изображений:
- Все тексты на русском, находятся прямо в .html файлах.
- Изображения заменяй в папке assets/images.
