# 📚 Система обучения языку "Забономӯзӣ" - Полная документация

## Обзор системы

Приложение "Забономӯзӣ" - это полнофункциональное приложение для изучения языков с прогрессивной системой геймификации, уровней и очков.

---

## 🎯 Основные компоненты

### 1. **Система прогресса букв** (Letter Progress System)

#### Класс `ProgressSystem`
Управляет всеми аспектами прогресса обучения.

```javascript
// Инициализация
const progress = new ProgressSystem();

// Методы работы с буквами
progress.markLetterComplete(lang, letter);      // Отметить букву как пройденную
progress.getLetterStatus(lang, letter);         // Получить статус: 'locked' | 'unlocked' | 'completed'
progress.getCompletedLetters(lang);             // Получить массив пройденных букв
```

#### Статусы букв:
- **locked** - буква недоступна (не пройдены предыдущие)
- **unlocked** - буква доступна для изучения
- **completed** - буква пройдена ✅

#### Хранение данных:
```javascript
localStorage.completedLetters = {
  'ru': ['А', 'Б', 'В', 'Г'],
  'en': ['A', 'B', 'C'],
  'tg': ['А', 'Б']
}
```

### 2. **Система очков и уровней** (Scoring System)

#### Методы системы очков:
```javascript
progress.addScore(points, message);   // Добавить очки с уведомлением
progress.userScore;                   // Текущий счёт
progress.getUserLevel();              // Получить объект уровня
```

#### Уровни:
| Уровень | Диапазон очков | Описание |
|---------|----------------|----------|
| 🟢 Новичок | 0-100 | Начальный уровень |
| 🔵 Ученик | 101-300 | Средний уровень |
| 🟣 Знаток | 301-600 | Продвинутый уровень |
| 🌟 Мастер | 600+ | Максимальный уровень |

#### Начисление очков:
- ✅ Пройденная буква: **+5 очков**
- 📝 Слогоделение (правильный ответ): **+10 очков**
- 🔤 Создание слова: **+20 очков**
- 🎉 Серия 3 слова подряд: **+50 бонус очков**
- 📖 Составление предложения: **+30 очков**

#### Хранение данных:
```javascript
localStorage.userScore = "350";  // Текущий счёт

localStorage.leaderboard = [
  { name: "Игрок1", score: 500, date: "22.04.2026" },
  { name: "Игрок2", score: 450, date: "22.04.2026" },
  // ... ещё 3 записи (TOP-5)
]
```

### 3. **Модуль слогоделения** (Syllable Module)

Игра на разбиение слов на слоги.

```javascript
startSyllableGame();  // Запустить игру
```

#### Требования:
- Минимум 1 пройденная буква
- Слова состоят из пройденных букв

#### Пример:
```
Слово: "ки | то | б"
Пользователь нажимает на слоги по очереди
Правильный ответ: +10 очков
```

#### Данные хранятся в:
```javascript
gameVocabulary.ru.syllables = [
  { word: 'ма-ма', phonetic: 'mama' },
  { word: 'па-па', phonetic: 'papa' },
  { word: 'ки-то-б', phonetic: 'kitob' },
  // ...
]
```

---

### 4. **Модуль создания слов** (Word Building Module)

Игра на составление слов из карточек букв.

```javascript
startWordBuildingGame();  // Запустить игру
```

#### Требования:
- Минимум 3 пройденные буквы
- Слова состоят из пройденных букв

#### Геймплей:
```
1. Буквы показываются в случайном порядке
2. Пользователь кликает по буквам в правильном порядке
3. Буквы складываются в слово
4. При 3 правильных словах подряд: +50 бонус очков
```

#### Очки:
- Правильное слово: **+20 очков**
- Серия 3 слова: **+50 бонус очков**

#### Система серий:
```javascript
wordBuildingStreak = 0;
// Увеличивается при каждом правильном слове
// Сбрасывается при ошибке
// При streak % 3 === 0: бонус +50 очков
```

---

### 5. **Модуль составления предложений** (Sentence Composition Module)

Игра на расставление слов в правильный порядок через Drag & Drop.

```javascript
startSentenceGame();  // Запустить игру
```

#### Требования:
- Минимум 5 пройденных букв
- Слова в предложениях состоят из пройденных букв

#### Геймплей:
```
1. Показывается перемешанное предложение
2. Слова можно перетаскивать (drag & drop)
3. Нужно расставить слова в правильный порядок
4. Правильный ответ: +30 очков
```

#### HTML структура:
```html
<div id="sentence-game">
  <div id="sentence-drop-zone"><!-- Зона для перетаскивания --></div>
  <div id="sentence-choices"><!-- Карточки со словами --></div>
  <div id="sentence-result"><!-- Результат --></div>
</div>
```

### 6. **Таблица рекордов** (Leaderboard)

Система сохранения и отображения топ-5 игроков.

```javascript
progress.updateLeaderboard(playerName);  // Добавить в рейтинг
progress.leaderboard;                    // Получить список TOP-5
```

#### Структура данных:
```javascript
{
  name: "Игроки Петров",
  score: 450,
  date: "22.04.2026"
}
```

## 🚀 Примеры использования

### Пример 1: Отметить букву как пройденную
```javascript
const letter = 'А';
const lang = 'ru';

progress.markLetterComplete(lang, letter);

// Буква добавится в:
// localStorage.completedLetters.ru = ['А']

// Пользователь получит +5 очков
progress.addScore(5, 'Буква пройдена: +5 очков');
```

### Пример 2: Запустить игру слогоделения
```javascript
// Проверяем, есть ли пройденные буквы
const completedLetters = progress.getCompletedLetters('ru');

if (completedLetters.length > 0) {
  startSyllableGame();  // Запускаем игру
} else {
  alert('Сначала пройдите буквы!');
}
```

### Пример 3: Получить текущий уровень
```javascript
const level = progress.getUserLevel();

console.log(level);
// { name: 'Ученик', level: 2, range: '101-300' }

console.log(progress.userScore);  // 150
```

### Пример 4: Обновить таблицу рекордов
```javascript
const playerName = 'Вы';
progress.updateLeaderboard(playerName);

// Система автоматически:
// 1. Добавит игрока в leaderboard
// 2. Отсортирует по очкам (убывающий порядок)
// 3. Оставит только TOP-5
// 4. Сохранит в localStorage
```

## 🔧 Важные функции

### updateLevelDisplay()
Обновляет отображение уровня и рекордов в меню.
```javascript
updateLevelDisplay();  // Вызывается автоматически при добавлении очков
```

### showScoreNotification(points, message)
Показывает уведомление о полученных очках.
```javascript
showScoreNotification(20, '✅ Слово создано: +20 очков');
```

### goMenu()
Возвращает в главное меню и обновляет статистику.
```javascript
goMenu();  // Автоматически вызывает updateLevelDisplay()
```
